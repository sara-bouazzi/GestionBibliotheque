# TU_WEB - Library Management

This is an ASP.NET Core project for a library management system, made for an exam. It lets users see who borrowed which books.

## What It Does
- Shows a list of book loans with names and titles (Index view).
- Lets you add some data manually (categories, users, books).

## How to Use
1. Download the project:
2. Open `TU_WEB.sln` in Visual Studio.
3. Set up the database with migrations.
4. Run it to see the loan list!

## Exam Part Done
- Question 9: Made the Index page to show loan details.

## Project Structure
- **Entity Framework**: Configured entities and migrations for the database.
- **ASP.NET MVC**: Implements the Index view for loan management.
- **Database**: Named `BibliothequeSaraBouazzi`.
